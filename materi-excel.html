Mahir Microsoft Excel: Dari Dasar hingga Macro/VBA

Pelajari cara mengolah data dengan cepat, membuat laporan interaktif menggunakan Pivot Table, hingga mengotomatisasi pekerjaan rutin dengan Macro dan VBA.

Modul 1: Pengenalan & Rumus Dasar

[ Tempatkan Embed Video YouTube/Drive di Sini ]

Membahas antarmuka Excel, format cell, dan rumus dasar aritmatika (SUM, AVERAGE, MIN, MAX, COUNT).

Modul 2: Fungsi Logika & Referensi (VLOOKUP/HLOOKUP)

Memahami penggunaan fungsi IF bertingkat, AND/OR, serta mengambil data dari tabel lain menggunakan VLOOKUP, HLOOKUP, dan XLOOKUP (Excel 365).

Modul 3: Pivot Table & Dashboard Data

Cara merangkum ribuan baris data dalam hitungan detik menggunakan Pivot Table dan memvisualisasikannya dengan Pivot Chart untuk membuat Dashboard.

Modul 4: Otomatisasi dengan Macro & VBA Dasar

Merekam aktivitas berulang dengan Record Macro dan memahami jendela Visual Basic for Applications (VBA) untuk mengedit script otomatisasi sederhana.
